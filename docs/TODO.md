## 地图数据

- [ ] **台湾省区域细化**：当前台湾省使用省级轮廓（`710000.json` 只有一个 feature），无法展示地级市/县级。DataV 接口 `710000_full.json` 返回 404，需寻找其他数据来源或手动制作 GeoJSON
- [ ] **海南省细节**：海南省的 GeoJSON 包含大量离岛（西沙群岛、南沙群岛等），这些区域会影响 `fitSize` 投影计算，导致海南本岛显示偏小。考虑过滤极远离岛或使用自定义投影范围
- [ ] **南海十段线**：当前过滤了 `100000_JD` 九段线 feature。可考虑单独渲染为虚线（不参与 `fitSize` 计算），以符合地图合规要求
