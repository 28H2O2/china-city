# 中国制城 — 中华历史风格设计稿

## 设计理念

以中国传统丹青地图、典籍装帧和青铜铭刻为视觉原型，融合宋代舆图的素雅与明清绢本地图的古朴，在现代 Web 界面中呈现"翻阅古籍地图"的沉浸感。

---

## 二、字体体系

```
标题：Noto Serif SC（思源宋体），bold，呈现古籍刻印质感
正文：Noto Sans SC（思源黑体），regular，兼顾可读性
备注/标签：Noto Serif SC，italic，仿注疏体例
```

标题显示示例：
```
中国制城                ← 思源宋体 Bold 24px
China City Level Map   ← Noto Sans SC 12px，字间距 0.15em
```

## 三、组件设计规范

### 3.1 顶部标题栏

```
┌─────────────────────────────────────────────────────────────┐
│  ◈ 中国制城                         [保存图片] [分享] [重置] │
│    China City Level Map                                     │
└─────────────────────────────────────────────────────────────┘
```

- 背景：竹黄 `#F5EDD6`，底部边框用 `#C8BAA0`（淡墨）
- 标题左侧加传统菱形装饰符 ◈（或用 SVG 绘制简化铜钱/回字纹图案）
- 按钮样式见下方

### 3.2 按钮设计

**主按钮（保存图片）**
```
背景：朱砂 #C0472A
文字：素白 #FAF6EC
边框：无
圆角：4px（方正，少圆润）
悬停：深朱 #9A3620
图标：细线风格（非 filled），与古典线描一致
内边距：8px 16px
```

**次要按钮（分享）**
```
背景：透明
文字：墨色 #2C2416
边框：1px solid #C8BAA0
圆角：4px
悬停：背景 #EDE5CF
```

**危险按钮（重置）**
```
背景：透明
文字：褐灰 #7A6850
边框：1px solid #C8BAA0
悬停：背景 #F0E5D5，文字 #C0472A
```

### 3.3 地图区域

- 地图容器背景：淡竹黄 `#EDE5CF`，四角加细框线（仿装裱）
- 地图 SVG 底色：`#E8F0F5`（天水蓝，仿古地图海洋）
- 省份描边：`#2C2416` 0.6px，hover 加深到 1px
- 省份 hover 效果：透明度降至 0.85，cursor pointer
- 四边添加细装饰框线（1px，颜色 `#C8BAA0`，内嵌 4px）

## 四、纹理与装饰细节

### 4.1 背景纹理
页面底色 `#F5EDD6` 上叠加极淡的纸纹 SVG pattern（noise 纹理，opacity 0.04），模拟宣纸质感：
```css
background-image: url("data:image/svg+xml,..."); /* 细纸纹 */
background-size: 200px 200px;
```

### 4.2 标题装饰
标题左侧用 CSS 绘制「回字纹」或「万字纹」小图标（纯 CSS border 叠加）。

### 4.3 地图装饰框
地图 SVG 区域四周加双线装饰边框（外框 1px，内框 0.5px，间距 4px），仿古地图装裱：
```css
.map-frame {
  border: 1px solid #C8BAA0;
  box-shadow: inset 0 0 0 4px #F5EDD6, inset 0 0 0 5px #C8BAA0;
}
```

### 4.4 面包屑导航（省级视图）
```
全国  ›  广东省
```
分隔符改用 `›`（单书名号风格），而非 `/`。

## 五、整体视觉基调

| 维度 | 描述 |
|------|------|
| 历史感来源 | 宋代舆图 + 明清绢本地图 + 传统典籍装帧 |
| 主要材质感 | 宣纸、绢布、竹简 |
| 核心色调 | 竹黄（底）+ 墨色（线条）+ 朱砂（强调）|
| 字体风格 | 宋体为主，兼用黑体保证可读性 |
| 装饰语言 | 回字纹、菱形、方正线框、印鉴圆点 |
| 整体感觉 | 典雅、沉稳、有历史厚度，非"复古滥俗" |

## 六、实现要点

1. **Google Fonts 引入**：`Noto Serif SC:400,700` + `Noto Sans SC:400`
2. **CSS 变量**：全面替换原有变量为古典色系
3. **Tailwind 自定义色彩**：在 `tailwind.config` 中定义 `bamboo`、`ink`、`zhusha` 等语义化颜色
4. **进度条**：去掉 `rounded-full`，改为 `rounded-none`（方角）
5. **按钮**：`rounded` 改为 `rounded-sm` 或 `rounded-none`
6. **边框**：统一使用 `border-[#C8BAA0]`
